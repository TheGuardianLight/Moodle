# Script d'installation de Moodle

## Compatibilité :

| Nom et version | Etat |
| :----- | ------: |
| Ubuntu 23.04 | ✅ |
| Debian 12 | ❌ |

## Instruction :

1. Télécharger le script Moodle.sh sur votre distribution Linux.
2. Rendre le script exécutable avec la commande suivante :
```chmod 711 moodle.sh```
3. Attendre que l'installation se termine.
